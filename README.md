# 📚 Anotações Básicas de HTML

## Estrutura básica de um documento HTML

```html
<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>

    <h1>My First Heading</h1>
    <p>My first paragraph.</p>

</body>
</html>
```

## Significado de cada elemento

- `<!DOCTYPE html>`: declaração que informa ao navegador que o documento utiliza a versão **HTML5**.
- `<html>`: elemento raiz do documento HTML, responsável por envolver todo o conteúdo da página.
- `<head>`: contém os metadados da página, como título, codificação de caracteres, estilos e scripts.
- `<title>`: define o título da página, exibido na aba ou na barra de título do navegador.
- `<body>`: define o corpo da página e contém todo o conteúdo visível ao usuário, como títulos, parágrafos, imagens, links, tabelas, listas e formulários.
- `<h1>`: define um título de nível 1, sendo o cabeçalho de maior importância da página.
- `<p>`: define um parágrafo de texto.

## Observação

> Alguns elementos HTML não possuem conteúdo e, por isso, não precisam de uma tag de fechamento. Esses elementos são chamados de **elementos vazios (empty elements)**.

Exemplos:

- `<br>` → quebra de linha.
- `<hr>` → linha horizontal.
- `<img>` → insere uma imagem.
- `<input>` → cria um campo de entrada em formulários.
- `<meta>` → fornece metadados sobre o documento.
- `<link>` → relaciona recursos externos, como arquivos CSS.
